Çoxluqlar Nəzəriyyəsi

Riyaziyyatın fundamental dillərindən biri olan çoxluqlar, obyektləri qruplaşdırmaq və onlar arasındakı əlaqələri anlamaq üçün universal bir vasitədir.

Çoxluq – müəyyən obyektlərin və ya elementlərin təkrarlanmayan bir toplusudur. Məsələn, sinifdəki şagirdlər, rənglər və ya ədədlər bir çoxluq təşkil edə bilər.

Çoxluqlar adətən böyük latın hərfləri ilə işarə olunur ($A, B, C, ...$), elementləri isə fiqurlu mötərizə `{}` daxilində göstərilir.

Bir 'a' obyektinin A çoxluğuna aid olması $a \in A$ kimi, aid olmaması isə $a \notin A$ kimi yazılır.

Püf Nöqtələri:

  • Sıra önəmli deyil: $\{1, 2, 3\}$ çoxluğu ilə $\{3, 1, 2\}$ çoxluğu eynidir.
  • Elementlər təkrarlanmır: $\{1, 2, 2, 3\}$ kimi yazılış qəbul olunmur, o, $\{1, 2, 3\}$ kimi başa düşülür.

Çoxluqların Təsvir Üsulları

1. Siyahı üsulu (Roster Method): Çoxluğun bütün elementləri sadalanır.

Nümunə: $A = \{\text{alma, armud, heyva}\}$

Nümunə: $B = \{2, 4, 6, 8\}$

2. Xarakteristik üsul (Set-Builder Notation): Elementlərin ortaq xassəsi göstərilir.

Nümunə: $B = \{x \mid x \text{ ikiyə bölünən tək rəqəmli natural ədəddir}\}$

3. Eyler-Venn diaqramları: Çoxluqlar və onlar arasındakı əlaqələr qapalı əyrilər (adətən dairələr) vasitəsilə təsvir edilir. Bu, məsələləri vizuallaşdırmaq üçün çox əlverişlidir.

Boş çoxluq: Heç bir elementi olmayan çoxluğa deyilir. $\emptyset$ və ya $\{\}$ kimi işarə olunur.

Sonlu və Sonsuz Çoxluqlar: Elementlərinin sayını natural ədədlə ifadə etmək mümkündürsə sonlu (məs, A={1,2,3}), mümkün deyilsə sonsuz çoxluqdur (məs, Natural ədədlər çoxluğu $N$).

Universal Çoxluq (Evrensel Küme): Verilmiş bir məsələnin kontekstində mümkün olan bütün elementləri özündə birləşdirən çoxluqdur və $U$ hərfi ilə işarə edilir.

Altçoxluq

Əgər $A$ çoxluğunun bütün elementləri eyni zamanda $B$ çoxluğunun da elementidirsə, $A$-ya $B$-nin altçoxluğu deyilir və $A \subseteq B$ kimi işarə olunur.

Nümunə: $A=\{1, 2\}$ və $B=\{1, 2, 3\}$ olarsa, $A \subseteq B$-dir.

Xüsusi altçoxluq: Əgər $A \subseteq B$ və $A \neq B$ olarsa, $A$-ya $B$-nin xüsusi altçoxluğu deyilir və $A \subset B$ kimi işarə olunur.

Altçoxluqlarla bağlı vacib qaydalar:

  • Boş çoxluq ($\emptyset$) istənilən çoxluğun altçoxluğudur.
  • Hər bir çoxluq özünün altçoxluğudur ($A \subseteq A$).
  • Elementlərinin sayı $n$ olan bir çoxluğun **$2^n$** sayda altçoxluğu var.
  • Elementlərinin sayı $n$ olan bir çoxluğun **$2^n - 1$** sayda xüsusi altçoxluğu var.

Nümunə: 3 elementi olan $A=\{a, b, c\}$ çoxluğunun $2^3 = 8$ sayda altçoxluğu var: $\emptyset, \{a\}, \{b\}, \{c\}, \{a,b\}, \{a,c\}, \{b,c\}, \{a,b,c\}$.

1. Çoxluqların Birləşməsi ($A \cup B$)

$A$ və $B$ çoxluqlarının bütün elementlərindən ibarət olan yeni çoxluğa onların birləşməsi deyilir. Məntiqi "və ya" (OR) kimi düşünülə bilər.

$$A \cup B = \{x \mid x \in A \text{ və ya } x \in B\}$$

Nümunə: $A=\{1, 2, 3\}$ və $B=\{3, 4, 5\}$ olarsa, $A \cup B = \{1, 2, 3, 4, 5\}$.

2. Çoxluqların Kəsişməsi ($A \cap B$)

Həm $A$, həm də $B$ çoxluğuna eyni zamanda daxil olan (yəni ortaq) elementlərdən ibarət olan çoxluğa onların kəsişməsi deyilir. Məntiqi "və" (AND) kimi düşünülə bilər.

$$A \cap B = \{x \mid x \in A \text{ və } x \in B\}$$

Nümunə: $A=\{1, 2, 3\}$ və $B=\{3, 4, 5\}$ olarsa, $A \cap B = \{3\}$.

Püf nöqtəsi: Əgər çoxluqların kəsişməsi boş çoxluqdursa ($A \cap B = \emptyset$), onlara kəsişməyən çoxluqlar deyilir.

3. Çoxluqların Fərqi ($A \setminus B$)

$A$ çoxluğunun $B$ çoxluğuna daxil olmayan elementlərindən ibarət olan çoxluğa $A$ və $B$ çoxluqlarının fərqi deyilir. "A-da olsun, amma B-də olmasın" deməkdir.

$$A \setminus B = \{x \mid x \in A \text{ və } x \notin B\}$$

Nümunə: $A=\{1, 2, 3\}$ və $B=\{3, 4, 5\}$ olarsa, $A \setminus B = \{1, 2\}$. Eyni zamanda, $B \setminus A = \{4, 5\}$.

4. Simmetrik Fərq ($A \Delta B$)

Yalnız $A$-ya və ya yalnız $B$-yə daxil olan elementlərdən ibarət çoxluğa simmetrik fərq deyilir. Başqa sözlə, birləşmədən kəsişmənin çıxılmasıdır.

$$A \Delta B = (A \cup B) \setminus (A \cap B) = (A \setminus B) \cup (B \setminus A)$$

Nümunə: $A=\{1, 2, 3\}$ və $B=\{3, 4, 5\}$ olarsa, $A \Delta B = \{1, 2, 4, 5\}$.

Çoxluğun elementlərinin sayı $n(A)$ kimi işarə olunur.

Element Sayı Düsturları (İçindən-Xaricdən Prinsipi)

  • İki çoxluğun birləşməsi üçün: $$n(A \cup B) = n(A) + n(B) - n(A \cap B)$$
  • Üç çoxluğun birləşməsi üçün: $$n(A \cup B \cup C) = n(A) + n(B) + n(C) - n(A \cap B) - n(A \cap C) - n(B \cap C) + n(A \cap B \cap C)$$
  • Fərq üçün: $$n(A \setminus B) = n(A) - n(A \cap B)$$
  • Simmetrik fərq üçün: $$n(A \Delta B) = n(A \cup B) - n(A \cap B) = n(A) + n(B) - 2 \cdot n(A \cap B)$$

Əməllərin Xassələri

  • Yerdəyişmə (Kommutativlik): $A \cup B = B \cup A$;    $A \cap B = B \cap A$
  • Qruplaşdırma (Assosiativlik): $(A \cup B) \cup C = A \cup (B \cup C)$
  • Paylanma (Distributivlik): $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
    $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

De Morgan Qanunları

$A'$ - $A$ çoxluğunun $U$ universal çoxluğuna tamamlayıcısıdır ($A' = U \setminus A$).

  • $(A \cup B)' = A' \cap B'$
  • $(A \cap B)' = A' \cup B'$
